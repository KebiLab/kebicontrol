# Commands

Wake word is configurable (default: `кеби`). Say the word, then the command.
Examples below assume the default wake word.

## Russian

### System
- `кеби выключи компьютер` / `кеби перезагрузи` / `кеби в сон`
- `кеби заблокируй` / `кеби выйди из учётки`

### Volume and brightness
- `кеби громче` / `кеби тише`
- `кеби громкость на 50`
- `кеби выключи звук` / `кеби включи звук`
- `кеби яркость на 70`

### Apps
- `кеби открой браузер` / `кеби открой дискорд` / `кеби открой стим`
- `кеби открой проводник` / `кеби открой настройки`
- `кеби закрой <имя>`

### Windows
- `кеби сверни всё` / `кеби разверни` / `кеби закрой окно`
- `кеби окно влево` / `кеби окно вправо`
- `кеби переключись на <имя>`

### Media
- `кеби пауза` / `кеби играй` / `кеби дальше` / `кеби назад`
- `кеби стоп`

### Input
- `кеби набери <текст>` — pastes text into the active window
- `кеби нажми enter` / `кеби нажми escape`
- `кеби кликни` / `кеби правый клик`

### Web
- `кеби найди <запрос>` — Google search
- `кеби открой сайт <url>` / `кеби открой ютуб` (alias)

### Screenshot
- `кеби скриншот`

### Timers
- `кеби таймер на 5 минут`
- `кеби напомни в 14:30 <текст>`

### Voice
- `кеби скажи <текст>` — TTS reply
- `кеби выключи голос` / `кеби включи голос`

### Dictation
- `кеби включи режим диктовки` / `кеби выключи диктовку`

### Help
- `кеби хелп` / `кеби помощь` — lists hotkeys

## English

- `kebi shutdown` / `kebi restart` / `kebi sleep`
- `kebi volume up` / `kebi volume down` / `kebi volume 50`
- `kebi open <name>` / `kebi close <name>`
- `kebi search <query>` / `kebi google <query>`
- `kebi screenshot` / `kebi pause` / `kebi play`
- `kebi type <text>` / `kebi press <key>`
- `kebi help`

## Hotkeys (default)

- `Ctrl+Shift+Space` — push-to-listen
- `Ctrl+Shift+K` — open overlay panel
- `Ctrl+Shift+M` — toggle voice replies
- `Ctrl+Shift+D` — dictation mode
- `Ctrl+Shift+P` — pause assistant
- `Esc` — cancel

All hotkeys are remappable in Settings.
